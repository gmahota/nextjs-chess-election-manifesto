## Fauna Cloud

### Install FaunaDB Shell

```
npm install -g fauna-shell
```

### Login to your FaunaDB account

```
fauna cloud-login
```

### Create Database

```
fauna create-key my_app
```

### Use Database

```
fauna shell my_app
```
